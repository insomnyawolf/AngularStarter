# AngularStarter

## Requisitos

* Tener instalado [Node](https://nodejs.org/)  
* Dos consolas  
* Cualquier editor de texto:  
* [VS Code](https://code.visualstudio.com/)  
* [Eclipse](https://www.eclipse.org/)  

### Instalar Node

```npm install -g @angular/cli```

### Crear Nuevo Proyecto 

```ng new (Nombre proyecto)```

### Iniciar servidor

```cmd
cd (Nombre proyecto)
ng serve --open
```
